# Crypto Dashboard

Projeto desenvolvido com HTML, CSS e JavaScript consumindo API pública da CoinGecko.

## Funcionalidades

- Listagem das 10 principais criptomoedas
- Consumo de API com fetch
- Uso de async/await
- Renderização dinâmica com JavaScript

## Tecnologias

- HTML
- CSS
- JavaScript
- API CoinGecko

## Objetivo

Projeto criado para praticar consumo de API e manipulação de dados dinâmicos no front-end.
